Импорт строк из акта

Введите 16-значный код акта рекламации, из которого хотите подгрузить данные по изделиям.

Импорт из excel файла

Вы можете загрузить данные по рекламации из файла в формате xlsx.
Загружаемый файл должен содержать следующие столбцы:
  • Серийный номер - в случае, если у вашего изделия отсутствует серийный номер, введите цифру 0.
  • Код RBZ - артикул товара
  • Наименование номенклатуры - укажите полное наименование изделия
  • Дата изготовления - в формате месяц.год (пример: 02.2022)
  • Срок эксплуатации - возможные значения: неисправность обнаружена сразу, до 1 года, от 1 до 2 лет, более 2 лет.
  • Условия эксплуатации - возможные значения: нормальные, высокая влажность, высокая температура, мелкодисперсная пыль, свой вариант.
  • Описание неисправности - опишите выявленную неисправность максимально подробно
